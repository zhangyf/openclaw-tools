# 标准日本语 初级上册 — 第6课知识点（学习版）

## 第6课「吉田さんは 来月 中国へ 行きます」— 去哪里、怎么去、和谁去

前5课学了"这是什么→在哪儿→有什么→几点做什么"，第6课学的是**移动**——去哪儿、怎么去、和谁去、从哪儿到哪儿。这是日语里最常用的日常表达之一。

### 本课核心：移动四件套

1. **去哪儿？** → 助词 **へ**（方向）
2. **怎么去？** → 助词 **で**（交通工具）
3. **和谁去？** → 助词 **と**（同伴）
4. **从哪到哪？** → **から〜まで**（起点〜终点）

---

### ① へ（移动方向）— "去往某地"

| 句型 | 意思 | 例子 |
|:----|:----|:----|
| 场所**へ** 行きます | 去某地 | 中国**へ** 行きます |
| 场所**へ** 来ます | 来某地 | 日本**へ** 来ました |
| 场所**へ** 帰ります | 回某地 | うち**へ** 帰ります |

**注意：** へ读作 **e**，不是he。虽然写是"へ"，发音是"え"。

> 跟第5课的 **に**（表时间）不同，へ专门表方向/目的地。

---

### ② で（交通工具）— "坐什么去"

| 交通方式 | 说法 |
|:--------|:----|
| 坐电车 | 電車**で** |
| 坐新干线 | 新幹線**で** |
| 坐飞机 | 飛行機**で** |
| 坐出租车 | タクシー**で** |
| 坐渡轮 | フェリー**で** |
| 走路 | 歩い**て**（步行是动词て形，不是で） |

> 歩いて（あるいて）是例外，用的是动词て形，不是名词+で。

**课文经典：**
> 森さんは 東京から 広島まで 新幹線**で** 行きます。
> 何**で** 駅へ 行きますか。（怎么去车站？）

---

### ③ と（同伴）— "和谁一起"

| 句型 | 意思 |
|:----|:----|
| 人**と** 行きます | 和某人去 |
| だれ**と** 行きますか | 和谁去？ |
| 友達**と** 行きます | 和朋友去 |

> 第4课学的**と**是"和"（并列，名1と名2），第6课的**と**表示"和某人一起做动作"。

---

### ④ から〜まで（起点〜终点）

第5课学过「時間から時間まで」表时间范围，第6课扩展到**地点范围**：

> 東京**から** 広島**まで** 新幹線で 行きます。
> 駅**から** アパート**まで** 歩いて帰りました。

**まとめて使う（组合使用）：**
> 駅から アパートまで 歩いて帰りました。
> （从车站走路回公寓——交通工具+方向+起终点全在一句话里）

---

### ⑤ 助词+は 对比用法

第5课学过单独的「は」表对比。第6课扩展为**助词+は**：

| 对比 | 例子 |
|:----|:----|
| **には** | わたしの部屋**には**電話がありません（我的房间没有电话——暗示其他地方可能有）|
| **へは** | 韓国**へは**行きました。中国**へは**行きませんでした（去了韩国，没去中国）|
| **からは** | 駅**からは**タクシーで帰りました（从车站开始打车——前面可能走路）|

---

### ⑥ 日期读法（1日〜10日特殊）

第5课学了月份和星期，第6课学**日期（几日）**。注意1〜10日有特殊读法：

| 日期 | 读法 | 日期 | 读法 |
|:---:|:----|:---:|:----|
| 1日 | **ついたち** | 6日 | むいか |
| 2日 | **ふつか** | 7日 | なのか |
| 3日 | **みっか** | 8日 | ようか |
| 4日 | **よっか** | 9日 | ここのか |
| 5日 | いつか | 10日 | **とおか** |
| | | 14日 | じゅう**よっか** |
| | | 20日 | **はつか** |
| | | 24日 | にじゅう**よっか** |
| | | 何日 | なんにち |

**规律：** 2〜10日用日本传统数法（和语数词），1日和20日比较特殊，14日和24日跟4日一样用よっか。

---

### 🗣 表达及词语讲解

#### たしか（大概/好像是）
不完全确定的记忆。课文里小李说"たしか11時半ごろです"（好像是11点半左右）。

#### うち（家）
课文多次出现「うちへ帰りました」。うち强调"我的家/自己家"，家(いえ)更偏指建筑本身。

#### まっすぐ（径直/直接）
> まっすぐ帰りました。（直接回家了。）

#### お疲れ様でした（辛苦了）
同事之间使用的寒暄语，对对方的辛苦表示感谢/慰劳。离开公司时也说「お先に失礼します」（我先走了）。

#### それは大変ですね（那真是不容易）
对大工作量表示同情/理解。

### 💬 应用课文「交通機関」

**场景：** 第二天早晨，小野搭话打瞌睡的森。

> **吉田：** 李さん、昨日は何時にアパートへ帰りましたか。
> **李：** ええと、たしか11時半ごろです。
> **吉田：** 何で帰りましたか。タクシーですか。
> **李：** 電車です。渋谷まで電車で行きました。駅からアパートまで歩いて帰りました。

> **小野：** 森さん、ゆうべはまっすぐ帰りましたか。
> **森：** いいえ、課長といっしょに銀座へ行きました。
> **李：** 2時ですか。それはお疲れ様でした。

---

### 一句话总结第6课

> 第5课学了"谁几点做什么"，第6课升级为**"谁从哪坐什么和谁一起去哪"**——一个完整的移动表达。へ(方向)・で(工具)・と(同伴)・から〜まで(起终点)，四个助词把一个动作交代得清清楚楚。
